# Security Policy

Use this template only on apps, devices, and systems you own or are explicitly authorized to test.

## Reporting issues

If you find a build, safety, or stability issue in the template, open a GitHub issue with:

- Android version and ABI
- LSPosed/Vector version
- Gradle command used
- Relevant logs with private package names redacted
- Steps to reproduce

Do not submit reports that include secrets, proprietary app code, paid content, bypass instructions for systems you are not authorized to test, or credentials.
